Cours : Introduction au HTML

Apprenez à créer vos premières pages web !

1. Structure de base d’une page HTML

Toute page commence par cette structure minimale :

Bienvenue sur ma page !

Voici un petit paragraphe.

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <title>Ma page</title>
  </head>
  <body>
    <h1>Bienvenue sur ma page !</h1>
    <p>Voici un petit paragraphe.</p>
  </body>
</html>

2. Titres et paragraphes

Les titres vont de <h1> à <h6> et les paragraphes utilisent <p>.

Titre principal

Sous-titre

Voici un paragraphe avec texte important.

<h1>Titre principal</h1>
<h2>Sous-titre</h2>
<p>Voici un paragraphe avec <span class="highlight">texte important</span>.</p>

3. Liens et images

Pour naviguer vers d'autres sites et ajouter des images :

Visiter Google
<a href="https://www.google.com" target="_blank">Visiter Google</a>
<img src="https://via.placeholder.com/150" alt="Exemple d'image">

4. Listes

Listes non ordonnée et ordonnée :

Liste non ordonnée :
  • Élément 1
  • Élément 2
Liste ordonnée :
  1. Premier
  2. Deuxième
<ul>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ul>
<ol>
  <li>Premier</li>
  <li>Deuxième</li>
</ol>

5. Petit bonus CSS

On peut styliser facilement :

Texte stylé avec CSS inline !

<p style="color: #ff9000; font-weight: bold;">Texte stylé avec CSS inline !</p>
