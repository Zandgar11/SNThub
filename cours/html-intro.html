Cours : Introduction au HTML

Les bases du HTML en SNT

1. Structure de base d’une page HTML

Chaque page HTML commence par une structure de base :

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Ma première page</title>
</head>
<body>
    <h1>Bienvenue</h1>
    <p>Voici un paragraphe.</p>
</body>
</html>

Explications :

  • <!DOCTYPE html> : indique que c’est un document HTML5.
  • <html lang="fr"> : enveloppe toute la page et indique la langue.
  • <head> : contient les métadonnées (titre, encodage…)
  • <body> : contient le contenu visible de la page.

2. Balises HTML essentielles

Titres

<h1>Titre principal</h1>
<h2>Sous-titre</h2>
<h3>Sous-sous-titre</h3>

Paragraphes et texte

<p>Ceci est un paragraphe.</p>
<p>On peut <strong>mettre en gras</strong> ou <em>en italique</em>.</p>
<span class="highlight">Texte important</span>

Liens

<a href="https://www.google.com" target="_blank">Visiter Google</a>

Images

<img src="monimage.jpg" alt="Description de l'image" width="300">

Listes

Liste non ordonnée :

<ul>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ul>

Liste ordonnée :

<ol>
  <li>Premier</li>
  <li>Deuxième</li>
</ol>

3. CSS de base

Le CSS permet de styliser votre contenu :

body {
    background-color: #f0f0f0;
    color: #333;
    font-family: 'Verdana', sans-serif;
}

p { line-height: 1.5; }

.highlight { color: #ff9000; font-weight: bold; }

4. Exemple complet de page

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Exemple HTML</title>
    <style>
        body { font-family: Arial; background: #f9f9f9; color: #333; }
        h1 { color: #2c3e50; }
        .highlight { color: #ff9000; font-weight: bold; }
    </style>
</head>
<body>
    <h1>Mon site web</h1>
    <p class="highlight">Texte important</p>
</body>
</html>
