Cours : Empêcher une image de se répéter en arrière-plan

Comprendre la propriété CSS background-repeat

1. Le problème : une image qui se répète

Lorsque tu mets une image en arrière-plan avec CSS, elle se répète automatiquement sur toute la zone. C’est le comportement par défaut de la propriété background-repeat.

Voici une image répétée (comportement par défaut) :

background-image: url("image.jpg"); background-repeat: repeat; /* Par défaut */

2. Empêcher la répétition : background-repeat: no-repeat;

Pour empêcher toute répétition, on utilise simplement :

background-repeat: no-repeat;

Voici une image qui ne se répète pas :

background-image: url("image.jpg"); background-repeat: no-repeat;

3. Centrer ou ajuster l'image

Souvent, quand on empêche la répétition, l’image reste dans le coin haut-gauche. Voici comment la repositionner :

  • Centrer : background-position: center;
  • Agrandir l’image pour remplir : background-size: cover;
  • Adapter sans couper : background-size: contain;

Image non répétée + centrée + redimensionnée :

background-image: url("image.jpg"); background-repeat: no-repeat; background-position: center; background-size: contain;

4. Résumé rapide

Ne pas répéter l'image : background-repeat: no-repeat;

La centrer : background-position: center;

La redimensionner : background-size: cover; ou contain;

/* Exemple complet */ div { background-image: url("image.jpg"); background-repeat: no-repeat; background-position: center; background-size: cover; }
